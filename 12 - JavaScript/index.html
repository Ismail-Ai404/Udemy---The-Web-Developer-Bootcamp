JS Basics

JavaScript Primitive Types

JavaScript has 7 primitive data types. These are immutable and represent single values.

1. String

Represents text. Strings are enclosed in single quotes (' '), double quotes (" "), or backticks (` `). 

let name = "Alice";
2. Number

Represents both integer and floating-point numbers. 

let age = 25;
let price = 99.99;
3. BigInt

Used for very large integers beyond Number limits. Created by adding n to the end. 

let big = 12345678901234567890n;
4. Boolean

Represents logical values: true or false. 

let isActive = true;
5. Undefined

A variable declared but not assigned any value has the value undefined. 

let x;
console.log(x); // undefined
6. Null

Represents an intentional absence of any value. It is an object type in JavaScript (historical quirk). 

let y = null;
7. Symbol

Introduced in ES6. Used to create unique identifiers for object properties. 

let id = Symbol('id');

Common JavaScript String Methods

Strings come with powerful built-in methods for manipulation and querying.

length

Returns the length of the string. 

let str = "Hello";
console.log(str.length); // 5
toUpperCase() & toLowerCase()

Convert a string to uppercase or lowercase. 

"hello".toUpperCase(); // "HELLO"
"HELLO".toLowerCase(); // "hello"
trim()

Removes whitespace from both ends of a string. 

"  hi  ".trim(); // "hi"
slice(start, end)

Extracts a part of the string. 

"JavaScript".slice(0,4); // "Java"
substring(start, end)

Similar to slice() but doesn't accept negative indexes. 

"Hello".substring(1,4); // "ell"
replace()

Replaces the first match with a new string. 

"I love cats".replace("cats", "dogs");
// "I love dogs"
includes()

Checks if a substring exists. 

"Hello world".includes("world"); // true
split()

Splits a string into an array based on a delimiter. 

"a,b,c".split(","); // ["a","b","c"]
JS Conditionals

JavaScript Booleans & Truthy/Falsy Values

Booleans represent logical values: true or false. Some values are inherently truthy and others are falsy.

Boolean Basics
let isActive = true;
let isComplete = false;

Falsy Values in JavaScript:

  • false
  • 0 and -0
  • "" (empty string)
  • null
  • undefined
  • NaN

Everything else is truthy (e.g., non-empty strings, numbers other than zero, objects, arrays).

Conditional Statements in JavaScript

Conditionals let you execute different blocks of code based on conditions.

if Statement

Executes a block of code if the condition is true. 

let age = 18;
if (age >= 18) {
  console.log("You are an adult.");
}
if...else Statement

Adds an alternative block if the condition is false. 

let age = 16;
if (age >= 18) {
  console.log("Adult");
} else {
  console.log("Minor");
}
else if Statement

Check multiple conditions in sequence. 

let score = 85;
if (score >= 90) {
  console.log("Grade A");
} else if (score >= 75) {
  console.log("Grade B");
} else {
  console.log("Grade C");
}
switch Statement

Selects a block to execute based on a matching case. 

let day = "Monday";
switch(day) {
  case "Monday":
    console.log("Start of the week");
    break;
  case "Friday":
    console.log("Weekend coming");
    break;
  default:
    console.log("Midweek");
}
Nested Conditions

You can put an if inside another if. 

let age = 20;
let hasID = true;

if (age >= 18) {
  if (hasID) {
    console.log("You can enter.");
  } else {
    console.log("ID required.");
  }
} else {
  console.log("Underage.");
}
