JS Basics

JavaScript Primitive Types

JavaScript has 7 primitive data types. These are immutable and represent single values.

1. String

Represents text. Strings are enclosed in single quotes (' '), double quotes (" "), or backticks (` `). 

let name = "Alice";
2. Number

Represents both integer and floating-point numbers. 

let age = 25;
let price = 99.99;
3. BigInt

Used for very large integers beyond Number limits. Created by adding n to the end. 

let big = 12345678901234567890n;
4. Boolean

Represents logical values: true or false. 

let isActive = true;
5. Undefined

A variable declared but not assigned any value has the value undefined. 

let x;
console.log(x); // undefined
6. Null

Represents an intentional absence of any value. It is an object type in JavaScript (historical quirk). 

let y = null;
7. Symbol

Introduced in ES6. Used to create unique identifiers for object properties. 

let id = Symbol('id');

Common JavaScript String Methods

Strings come with powerful built-in methods for manipulation and querying.

length

Returns the length of the string. 

let str = "Hello";
console.log(str.length); // 5
toUpperCase() & toLowerCase()

Convert a string to uppercase or lowercase. 

"hello".toUpperCase(); // "HELLO"
"HELLO".toLowerCase(); // "hello"
trim()

Removes whitespace from both ends of a string. 

"  hi  ".trim(); // "hi"
slice(start, end)

Extracts a part of the string. 

"JavaScript".slice(0,4); // "Java"
substring(start, end)

Similar to slice() but doesn't accept negative indexes. 

"Hello".substring(1,4); // "ell"
replace()

Replaces the first match with a new string. 

"I love cats".replace("cats", "dogs");
// "I love dogs"
includes()

Checks if a substring exists. 

"Hello world".includes("world"); // true
split()

Splits a string into an array based on a delimiter. 

"a,b,c".split(","); // ["a","b","c"]
JS Operators

JavaScript Comparison Operators

These operators are used to compare values and determine equality or inequality in JavaScript.

=== (Strict Equality)

Compares both value and type. Returns true only if both are identical. 

console.log(5 === 5);    // true
console.log(5 === "5");  // false
!== (Strict Inequality)

True if value or type is different. 

console.log(5 !== 5);    // false
console.log(5 !== "5");  // true
== (Loose Equality)

Compares values only, performs type coercion if necessary. 

console.log(5 == "5");   // true
console.log(null == undefined); // true

Warning: Type coercion can lead to unexpected results, so use with caution.

!= (Loose Inequality)

Checks if values are not equal, with type coercion. 

console.log(5 != "5");   // false
console.log(0 != false); // false (because 0 is coerced to false)

Key Differences Between Strict and Loose Equality

  • === / !==: No type conversion; compares type and value (strict).
  • == / !=: Performs type coercion; only compares value (loose). 
console.log(0 == false);  // true (coerced)
console.log(0 === false); // false (different types)
JS Booleans & Conditionals

JavaScript Booleans & Truthy/Falsy Values

Booleans represent logical values: true or false. Some values are inherently truthy and others are falsy.

Boolean Basics
let isActive = true;
let isComplete = false;

Falsy Values in JavaScript:

  • false
  • 0 and -0
  • "" (empty string)
  • null
  • undefined
  • NaN

Everything else is truthy (e.g., non-empty strings, numbers other than zero, objects, arrays).

Conditional Statements in JavaScript

Conditionals let you execute different blocks of code based on conditions.

if Statement

Executes a block of code if the condition is true. 

let age = 18;
if (age >= 18) {
  console.log("You are an adult.");
}
if...else Statement

Adds an alternative block if the condition is false. 

let age = 16;
if (age >= 18) {
  console.log("Adult");
} else {
  console.log("Minor");
}
else if Statement

Check multiple conditions in sequence. 

let score = 85;
if (score >= 90) {
  console.log("Grade A");
} else if (score >= 75) {
  console.log("Grade B");
} else {
  console.log("Grade C");
}
switch Statement

Selects a block to execute based on a matching case. 

let day = "Monday";
switch(day) {
  case "Monday":
    console.log("Start of the week");
    break;
  case "Friday":
    console.log("Weekend coming");
    break;
  default:
    console.log("Midweek");
}
Nested Conditions

You can put an if inside another if. 

let age = 20;
let hasID = true;

if (age >= 18) {
  if (hasID) {
    console.log("You can enter.");
  } else {
    console.log("ID required.");
  }
} else {
  console.log("Underage.");
}
JS Arrays

JavaScript Arrays

Arrays are used to store multiple values in a single variable. They are reference types in JavaScript.

Declaring Arrays with const

We often use const for arrays because it prevents reassignment of the variable name, but we can still modify the contents. 

const fruits = ["Apple", "Banana", "Cherry"];
// You can modify the array:
fruits.push("Orange");
// But you cannot do this:
fruits = ["New Array"]; // ❌ Error

Common Array Methods

push()

Adds an element to the end of an array. 

const arr = [1, 2];
arr.push(3); // [1, 2, 3]
pop()

Removes the last element and returns it. 

const arr = [1, 2, 3];
arr.pop(); // [1, 2]
unshift()

Adds an element to the start of an array. 

const arr = [2, 3];
arr.unshift(1); // [1, 2, 3]
shift()

Removes the first element and returns it. 

const arr = [1, 2, 3];
arr.shift(); // [2, 3]
concat()

Merges two or more arrays into a new array. 

const a = [1, 2];
const b = [3, 4];
const c = a.concat(b); // [1, 2, 3, 4]
indexOf()

Finds the index of a value, or returns -1 if not found. 

const arr = ["a","b","c"];
arr.indexOf("b"); // 1
includes()

Checks if an array contains a value.

const arr = ["x","y"];
arr.includes("y"); // true
reverse()

Reverses the array in place.

const arr = [1, 2, 3];
arr.reverse(); // [3, 2, 1]
slice()

Returns a shallow copy of a portion of the array. 

const arr = [1, 2, 3, 4];
arr.slice(1,3); // [2, 3]
splice()

Adds/removes items from an array at a specific index. 

const arr = [1, 2, 4];
arr.splice(2, 0, 3); // [1, 2, 3, 4]

Multidimensional Arrays

An array of arrays can represent matrices or grids. 

const matrix = [
  [1, 2, 3],
  [4, 5, 6],
  [7, 8, 9]
];
console.log(matrix[1][2]); // 6

Reference Types & Equality

Arrays are reference types, meaning variables store a reference to the object, not the actual value. 

const arr1 = [1, 2];
const arr2 = [1, 2];
console.log(arr1 === arr2); // false (different references)

const arr3 = arr1;
console.log(arr1 === arr3); // true (same reference)

Two separate arrays with the same values are not equal because they occupy different memory addresses.

JS Objects

JavaScript Object Literals

An object literal in JavaScript is a way to define an object using curly braces {}. Objects store data in key-value pairs and can hold numbers, strings, arrays, functions, or other objects.

Creating an Object Literal
const user = {
  name: "Alice",
  age: 25,
  isAdmin: true
};
Accessing Object Data

You can use dot notation or bracket notation. 

console.log(user.name);       // "Alice"
console.log(user["age"]);     // 25
Modifying Objects

We can add, update, or delete properties:

user.age = 26;        // Update value
user.email = "a@example.com"; // Add new property
delete user.isAdmin;  // Remove property

Real-World Example: Nested Objects & Arrays

Here’s a simplified data structure for a Reddit-like application: 

const redditData = {
  community: "JavaScript",
  posts: [
    {
      id: 101,
      title: "Understanding Closures",
      author: { username: "dev_guru", karma: 1200 },
      comments: [
        { user: "code_lover", text: "Great explanation!", upvotes: 15 },
        { user: "js_fan", text: "Thanks for sharing!", upvotes: 10 }
      ]
    },
    {
      id: 102,
      title: "ES2025 Features",
      author: { username: "future_js", karma: 800 },
      comments: [
        { user: "alice", text: "Excited for this!", upvotes: 22 },
        { user: "bob", text: "Can't wait to try them!", upvotes: 18 }
      ]

  ]
};

Extracting Specific Information

Example: Get the text of the first comment on the first post: 

const firstComment = redditData.posts[0].comments[0].text;
console.log(firstComment); // "Great explanation!"

Example: Get the username of the author of the second post: 

const secondAuthor = redditData.posts[1].author.username;
console.log(secondAuthor); // "future_js"
JS Loops & Practice

JavaScript Loops

Learn loops, iteration, and apply them in real-world mini projects.

1. Basic Loops

For Loop:

for (let i = 0; i < 5; i++) {
  console.log(i);
}

While Loop:

let count = 0;
while (count < 5) {
  console.log(count);
  count++;
}

Nested Loops Example:

for (let i = 1; i <= 3; i++) {
  for (let j = 1; j <= 2; j++) {
    console.log(`i=${i}, j=${j}`);
  }
}

Infinite Loop: Be careful! Example: 

// while(true) { console.log("Endless"); }

2. Break & Continue

break exits the loop, continue skips current iteration. 

for (let i = 0; i < 5; i++) {
  if (i === 3) break;   // stop when i = 3
  if (i === 1) continue; // skip 1
  console.log(i);
}

3. Iterating Arrays & Objects

// For arrays
const arr = ["a","b","c"];
for (let item of arr) console.log(item);

// For objects
const obj = {name:"Alice", age:25};
for (let key in obj) console.log(key, obj[key]);

Note: Use for...of for arrays, for...in for object keys.

4. forEach Loop

const nums = [1,2,3];
nums.forEach(num => console.log(num * 2));
// Difference: cannot break or continue easily
