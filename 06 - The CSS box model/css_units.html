CSS Unit Types

CSS supports various unit types for defining sizes and lengths. Here are some common ones:

  1. px: Pixels, a fixed unit.
  2. em: Relative to the font size of the element.
  3. rem: Relative to the font size of the root element.
  4. %: Percentage of the parent element's size.
  5. vh: Viewport height, 1% of the viewport height.
  6. vw: Viewport width, 1% of the viewport width.

These units can be used in various CSS properties like width, height, margin, padding, etc.

For example:

          .box {
               width: 50%; /* 50% of the parent element's width */
               height: 100px; /* Fixed height of 100 pixels */
               padding: 1em; /* Padding relative to the font size */
               margin: 2rem; /* Margin relative to the root font size */
          }

Using these units allows for responsive design and better adaptability to different screen sizes.
