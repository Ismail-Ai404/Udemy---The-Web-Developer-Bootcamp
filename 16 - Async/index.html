Quick facts
  • Call Stack ‚Äî where functions run (LIFO). Top item is currently executing.
  • Web APIs ‚Äî browser timers, fetch, DOM events. Work off the main stack.
  • Microtask Queue ‚Äî promise .then & queueMicrotask; processed after current stack, before macrotasks.
  • Task (macrotask) Queue ‚Äî setTimeout, setInterval, UI events; processed after microtasks.
