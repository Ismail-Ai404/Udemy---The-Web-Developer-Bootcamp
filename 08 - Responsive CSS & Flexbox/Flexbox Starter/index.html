Let's Play With Flexbox

This is a playground to experiment with Flexbox properties. You can modify the CSS in the app.css file to see how the layout changes.

Flex Direction

The flex-direction property defines the direction in which the flex items are placed in the flex container. You can change it to see how the layout of the boxes changes. Can be in the x-axis (row) or y-axis (column) and can be reversed.

Change the flex-direction property in the CSS to see how it affects the layout of the boxes. flex-direction can be set to row, column, row-reverse, or column-reverse.

Justify Content

The justify-content property defines how the flex items are aligned along the main axis. You can change it to see how the alignment of the boxes changes. It can be either at the start (default), end, center, spaced out or spaced out evenly.

Change the justify-content property in the CSS to see how it affects the alignment of the boxes. You can set it to flex-start, flex-end, center, space-between, or space-around.

Flex Wrap

The flex-wrap property defines whether the flex items should wrap onto multiple lines. You can change it to see how the layout changes when items wrap.

Change the flex-wrap property in the CSS to see how it affects the layout of the boxes. You can set it to nowrap, wrap, or wrap-reverse.

Align Items

The align-items property defines how the flex items are aligned along the cross axis. You can change it to see how the alignment of the boxes changes.
